========================================================================
    アプリケーション : GIO プロジェクトの概要
========================================================================

AppWizard はユーザー用に GIO アプリケーションを作成しました。

このファイルには GIO アプリケーションを構成している
各ファイルの概要説明が含まれています。

GIO.vcproj
    これはアプリケーション ウィザードで生成された VC++ プロジェクトのメイン プロジェクト ファイルです。
    このファイルには、ファイルを生成した Visual C++ のバージョン、プラットフォーム、
    およびアプリケーション ウィザードで選択されたプロジェクトの機能に関する情報が
    含まれています。

Form1.cpp
    このファイルはメイン アプリケーションのソース ファイルです。
    フォームを表示するコードを含んでいます。

Form1.h
    フォーム クラスと InitializeComponent() 関数の実装を含んでいます。

AssemblyInfo.cpp
    アセンブリ メタデータを編集するためのカスタム属性を含んでいます。

/////////////////////////////////////////////////////////////////////////////
その他の標準ファイル :

StdAfx.h, StdAfx.cpp
    これらのファイルはプリコンパイル済み (PCH) ヘッダー ファイル Win32.pch
    およびプリコンパイルされた型のファイル StdAfx.obj をビルドするために使われます。
