Minimal Board Editor (MBE) Version 0.51.14
                                                2012/12/25
1. 概要
-------
  MBEは、プリント基板編集ソフトウェアです。
  実行には、.NET Framework 2.0 以上が必要になります。
  Windows XP/Vista/7で動作します。

  300mm×300mmまでの片面/両面/4層基板の設計ができます。
  プリントアウトして感光基板の版下を作ったり、基板製造業者さんに
発注するためのガーバーデータを出力することができます。

  * プリント基板の製造を業者さんにお願いすると、それなりのお金が
    かかります。このソフトウェアが出力したガーバーデータでプリント
    基板をお作りになる場合はガーバービューワなどで十分なチェックを
    行ってください。ガーバービューワには無償で優秀なものがネットに
    あります。
  * このソフトウェアを使って、あなたがどんな損害を蒙っても作者は
    責任を負いません。

  配布ファイルは Visual Stusio 2008 のソリューションフォルダを
そのまま圧縮したもので、すべてのソースコードを含んでいます。
  私が使っているのは、Visual Stusio 2008 Standard Edition ですが
Visual C# 2008 Express Edition でもビルドできます。
  ソースコードの取り扱いについては、下記「4. 本ソフトウェアについて
の最新情報とライセンス」をご覧ください。


2. インストール
---------------
  あらかじめ .NET Framework 2.0 以上をWindows Updateでインストール
  しておいてください。(Vistaには初めからインストールされています)
  ↓.NET Frameworkのインストールの方法について
  http://www.suigyodo.com/online/mbe/manual/install/dotnetinst.htm
  
  MBEの実行に必要なファイルは、mbe\bin\Releaseフォルダのmbe.exeと、
toolmark.mb3です。この2つのファイルを適当な同じフォルダにコピーして、
必要に応じてmbe.exeのショートカットなど作ってください。
  
  PCを起動した直後の1回目の実行には時間がかかりますが、2回目以降は
それほどでもありません。
